# 編集可能な歴史年表（テスト中）

## 使い方

- 1.[ここをクリック](https://sotomono.github.io/editable-historical-timeline/)してページに飛びます

- 2.ページの中の「この年に出来事を追加する」を押します

- 3.詳細を入力して「記録する」を押すとその出来事が年表に反映されます

- 4.自由に自分だけの年表をつくろう！



## 注意！

- 編集した年表は、ページを開いている間だけ保存されています。再読み込みなどを行なうと記録が消えてしまいます！

- 編集した年表を保存する時はページ下部の「現状の年表をダウンロードする」をクリックしてください。

- 今の年表の状態を自分の機器にダウンロードできます。
